# StarSpeak 使用指南

## 概述

StarSpeak 是一个个人英语学习系统，提供离线查词、AI助助、自由对话和智能练习等功能。

## 系统要求

- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 网络连接（用于AI功能和同步）
- JavaScript 已启用

## 快速开始

### 1. 创建账户

1. 打开应用
2. 点击"注册"（Sign Up）选项卡
3. 填写以下信息：
   - 用户名：自定义用户名
   - 邮箱：有效的电子邮件地址
   - 密码：至少 6 个字符
4. 点击"创建账户"按钮
5. 注册成功后，自动跳转到主页

### 2. 登录

1. 打开应用
2. 在"登录"（Login）选项卡中输入：
   - 邮箱地址
   - 密码
3. 点击"登录"按钮

### 3. 语言切换

1. 登录后，在顶部栏找到语言切换按钮
2. 点击"EN"切换为英文
3. 点击"中文"切换为中文
4. 语言偏好会自动保存

## 主要功能

### 1. 查词（Lookup）
**目的**：快速查询英语单词的含义

**操作步骤**：
1. 点击底部导航栏的"查词"（查词）图标
2. 在搜索框输入要查询的英文单词
3. 查看单词的以下信息：
   - 音标（Phonetic）
   - 中文翻译（Translation）
   - 词性和定义（Definition）
   - 例句（Example）
4. 点击"加入练习"按钮将单词添加到练习库

### 2. SOS 紧急求助
**目的**：快速获取英语问题的解答

**操作步骤**：
1. 点击底部导航栏的"SOS"图标
2. 输入你的问题（英文或中文）
3. 点击"发送"按钮
4. AI 将提供详细的解答和解释

### 3. 练习（Drill）
**目的**：通过循环练习巩固词汇记忆

**操作步骤**：
1. 点击底部导航栏的"练习"（Drill）图标
2. 系统会显示添加到练习库的单词
3. 查看单词信息并尝试回答问题
4. 提交答案，获得反馈
5. 继续下一个单词

**提示**：
- 定期参加练习可以帮助你更好地记忆单词
- 系统会跟踪你的学习进度

### 4. 自由对话（Free Talk）
**目的**：与 AI 进行自由英语对话

**操作步骤**：
1. 点击底部导航栏的"对话"（Talk）图标
2. 在消息框输入你的内容
3. 点击"发送"按钮
4. AI 将响应并继续对话
5. 可以进行多轮对话

## 界面导航

### 顶部栏
- **用户头像和邮箱**：显示当前登录的账户信息
- **语言切换**：选择应用语言（EN / 中文）
- **登出按钮**：退出当前账户

### 底部导航栏
四个主要功能入口：
- **SOS**（求助）- 红色梯度背景
- **查词**（词典） - 蓝色梯度背景
- **练习**（健身）- 紫色梯度背景
- **对话**（聊天） - 蓝紫色梯度背景

点击任意按钮即可切换到对应功能。

## 常见问题

### Q: 我忘记了密码怎么办？
A: 目前版本不支持密码重置。请使用正确的邮箱和密码登录。如需帮助，请重新注册新账户。

### Q: 我的学习数据会被保存吗？
A: 是的。所有数据都存储在 Supabase 数据库中，与你的账户绑定。只要你登录，就能访问所有数据。

### Q: 支持离线使用吗？
A: 词典数据可以离线查询。但是 SOS、对话和练习反馈功能需要网络连接来与 AI 交互。

### Q: 我可以删除我的账户吗？
A: 目前版本不支持直接删除账户。请联系管理员帮助处理。

### Q: 如何报告错误或提建议？
A: 请在应用中使用"反馈"功能或联系开发者。

## 学习建议

1. **定期练习**：每天花 20-30 分钟进行练习
2. **主动使用**：在查词功能中积极添加新单词
3. **参与对话**：充分利用 SOS 和自由对话来提高英语水平
4. **跟踪进度**：定期检查你的练习结果和学习历史
5. **定期复习**：复习已学过的单词和短语

## 数据安全

- 所有通信都通过 HTTPS 加密
- 密码存储使用 Supabase Auth 的安全机制
- 个人数据仅用于学习目的
- 遵守隐私政策

## 技术支持

如遇到问题：
1. 确保浏览器是最新版本
2. 清除浏览器缓存并重新加载
3. 检查网络连接
4. 尝试使用其他浏览器
5. 如问题仍未解决，请联系开发者
